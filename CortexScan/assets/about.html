CortexScan™ is an application that demonstrates the power of Code's proprietary barcode decoder, CortexDecoder®.

CortexDecoder® is a powerful decode platform that can scan all 1D and 2D barcodes, including UPC, Code 39, Code 128, QR Code, Data Matrix, and many more. CortexDecoder® is available for integration into Android applications via an easy-to-use Software Development Kit that can be obtained by contacting Code at www.tachyoniq.com/contact-us/.
